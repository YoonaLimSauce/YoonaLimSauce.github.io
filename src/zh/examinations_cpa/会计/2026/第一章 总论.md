# 总论

## 会计概述

### 会计定义

经济管理工作, 以货币为主要计量单位

### 会计人员职业道德

1. 坚持诚信, 守法奉公
2. 坚持准则, 守责敬业
3. 坚持学习, 守正创新

职业道德基本准则首要地位: 诚信

## 财务报告目标、会计基本假设和会计基础

### 会计基本假设

1. 会计主体
2. 持续经营
3. 会计分期
4. 货币计量

### 会计基础

1. 权责发生制
2. 收付实现制
    - 政府会计的预算会计

## 会计信息质量要求

1. 可靠性
2. 相关性
3. 可比性
4. 及时性
5. 实质重于形式
6. 重要性
7. 谨慎性
8. 可理解性

## 会计要素及其确认与计量

### 会计要素

1. 资产
2. 负债
3. 所有者权益
4. 收入
5. 费用
6. 利润

### 会计要素计量属性

1. 历史成本
2. 重置成本
3. 可变现净值
4. 现值
5. 公允价值
    - 计量日, 有序交易市场, 出售一项资产所能收到或者转移一项负债所需支付的价格

## 可持续信息

### 可持续信息披露的目标与原则

重要性原则
    - 财务重要性
    - 影响重要性

### 可持续信息质量要求

1. 可靠性
2. 相关性
3. 可比性
4. <strong style="color: red">可验证性</strong>
5. 可理解性
6. 及时性

1. 治理
2. 战略
3. 风险和机遇管理
4. 指标和目标
